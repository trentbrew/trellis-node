Trellis Realtime

Multiplayer demos

Four primitives built on the Trellis realtime layer — ephemeral presence and broadcast that never touches the persisted causal log. Default mode runs in this page across simulated peers; live relay mode syncs real browser windows through the demo server.

Mode: simulated room (peers in this page) Switch to live relay ↗ Live mode relays through ws://…/rt (all browsers). Chat is a grow-only set keyed by message id: each browser keeps its own replica (this tab's localStorage) and the relay keeps another. They converge on reconnect — so a refresh restores instantly and even survives a relay restart on this browser. The shared document persists for the server session. Same origin everywhere (localhost, not 127.0.0.1).
