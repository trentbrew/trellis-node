# Idea Garden

The Idea Garden preserves abandoned or interrupted work as reusable context.

## Inspect garden state

```bash
trellis garden list
trellis garden search -k "sync"
trellis garden stats
```

## Revive work

```bash
trellis garden revive <cluster-id>
```

The Garden matters for agentic work because context switches are normal. Trellis should make those discarded paths searchable rather than lost.
