# Decision Traces

Decision traces make reasoning inspectable.

A useful agent runtime should store more than final outputs. It should preserve tool calls, rationale, alternatives, prompt context, affected entities, and resulting mutations.

## CLI usage

```bash
trellis decision list
trellis decision show DEC-1
trellis decision chain issue:TRL-7
```

## Why it matters

When an AI system changes state, a user should be able to ask why and traverse the graph to inspect the answer.
