# Server

Import server APIs from `trellis/server`.

```ts
import { startServer, TenantPool, FULLY_PUBLIC } from "trellis/server";
```

The server is a compatibility and access surface over Trellis state. It should not be treated as the only possible source of truth.

## Current capabilities

- HTTP entity CRUD
- EQL-S query endpoint
- Blob upload/download
- Auth helpers
- Permission registry
- Tenant pool
- WebSocket realtime subscriptions
- Built-in inspector route

## Realtime behavior

Clients subscribe to EQL-S query strings over WebSocket. After HTTP create/update/delete, the server notifies subscriptions by re-running queries and sending full results plus diffs.
