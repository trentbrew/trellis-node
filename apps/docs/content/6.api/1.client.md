# Client SDK

Import the SDK from `trellis/client`.

```ts
import { TrellisClient } from "trellis/client";
```

## Local mode

```ts
const db = new TrellisClient({ path: "./.trellis-db" });

const id = await db.create("Task", { title: "Write docs", status: "open" });
const task = await db.read(id);
await db.update(id, { status: "done" });
const tasks = await db.list("Task");
const result = await db.query('find Task where status = "done"');

console.log(task, tasks.data, result.bindings);
db.close();
```

## Remote mode

```ts
const db = new TrellisClient({
  url: "http://localhost:3000",
  apiKey: "your-api-key",
  tenantId: "default",
});
```

Remote mode supports CRUD, query, upload/download, auth helpers, and WebSocket query subscriptions.

```ts
const sub = db.subscribe('find Task where status = "open"', (rows, diff) => {
  console.log(rows, diff.added, diff.updated, diff.removed);
});

sub.unsubscribe();
db.disconnect();
```

Local live subscriptions are part of the runtime roadmap.
