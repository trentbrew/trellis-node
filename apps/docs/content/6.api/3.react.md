# React Bindings

Import React bindings from `trellis/react`.

```ts
import { TrellisProvider, useQuery, useMutation } from "trellis/react";
```

## Current hooks

- `TrellisProvider`
- `useTrellis`
- `useEntity`
- `useEntities`
- `useQuery`
- `useMutation`

`useQuery` uses `client.subscribe` for live EQL-S query updates in remote mode.

## Direction

React bindings should remain thin adapters over a framework-agnostic Trellis client/runtime. TSRX, Svelte, Vue, Solid, and other framework integrations should bind to the same core primitives rather than owning sync semantics.
