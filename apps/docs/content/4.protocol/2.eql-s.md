# EQL-S

EQL-S is the structured query language used by the Trellis query engine.

## Simple query form

```text
find <Type> where <attribute> = "<value>"
```

Example:

```text
find Note where pinned = "true"
```

From the SDK:

```ts
const result = await db.query('find Note where pinned = "true"');
console.log(result.bindings);
```

## Direction

EQL-S should support both application views and provenance queries. The same query model should power UI projections, agent inspection, research corpus exploration, and runtime diagnostics.
