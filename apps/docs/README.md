# Trellis Docs

This is the Nuxt Content documentation site for Trellis.

Trellis is a local-first agentic OS runtime: an owned causal graph substrate for auditable agents, programmable views, peer sync, and machine-readable knowledge lifecycles.

## Source of truth

- Canonical source docs live in `../../docs`.
- Published website content lives in `content`.
- The reusable Nuxt shell was migrated from `trellis-docs/www`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
